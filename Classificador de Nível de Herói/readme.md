**Classificateur de Niveau de Héros**

Description

Ce projet est un classificateur de niveau pour les héros basé sur leur expérience (XP). Il utilise des variables, des opérateurs, des boucles de répétition et des structures de décision en JavaScript pour déterminer le niveau d'un héros.

Niveaux

    Moins de 1000 XP : Fer
    1001-2000 XP : Bronze
    2001-5000 XP : Argent
    5001-7000 XP : Or
    7001-8000 XP : Platine
    8001-9000 XP : Ascendant
    9001-10000 XP : Immortel
    Plus de 10001 XP : Rayonnant

Comment utiliser

    Clonez ce dépôt sur votre machine.
    Ouvrez un terminal et naviguez vers le dossier du projet.
    Exécutez la commande node index.js.
    La sortie affichera le niveau du héros défini dans le code (par exemple, Aragorn avec 7500 XP).

Technologies utilisées

    JavaScript
    Node.js
